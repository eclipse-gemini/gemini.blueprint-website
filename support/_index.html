Community Forum

The Gemini Forum is intended for user community discussions on Gemini projects (including Blueprint) usage and troubleshooting. Developers extending Blueprint should refer to the contribute page and the Gemini Developer mailing list.

Open Source Support for Gemini Blueprint

Submit bugs using the Bugzilla web interface. Please search for duplicates first.

For defects, provide steps to reproduce. For enhancements, describe a use case.

Committers prioritize critical bug fixes and enhancements and cannot respond to all inquiries. For additional help, see the Commerical Support section below.

  • View bugs with severity of normal or higher
  • View bugs marked enhancement, minor, or trivial

Commercial Support for Gemini Blueprint

     SpringSource the originator of Gemini Blueprint and Virgo provides commercial support at the Basic and Production levels. This is a separate support offering and is not tied to any other Spring projects or offerings.
